# Internal documentation

Some stunt documentation is kept in a **private** repository,
[`stuntapi/stunt-internal`](https://github.com/stuntapi/stunt-internal), because
it contains competitive research, internal roadmap notes, or references to
other (unpublished) projects.

This includes:

- `docs/research/` — competitive & market research
- `docs/plans/` — internal epic/roadmap plans
- `docs/dogfood/` — dogfooding findings and integration notes
- `AGENTS.md` — internal agent configuration

Access is restricted to the maintainers. The public
[`stuntapi/stunt`](https://github.com/stuntapi/stunt) repository contains only
what is intended to be public.

If you are looking for public design or usage documentation, see:

- [Design spec](specs/) — architecture and component design
- [CONTRIBUTING.md](../CONTRIBUTING.md) — contributor guide
- [RELEASING.md](../RELEASING.md) — release & distribution setup
- [README.md](../README.md) — project overview
